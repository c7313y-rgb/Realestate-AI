# 不動産AIエージェント SaaS UIデモ

GitHub Pagesでそのまま公開できる静的Webアプリです。添付UIをベースに、デモ版でも主要機能を実際に操作できるよう強化しています。

## デモ版で体験できる機能
- 左サイドバー型SaaSダッシュボード
- OpenStreetMap + Leafletの物件マップ、フィルタ、物件詳細ドロワー
- AI収益予測・KPIカード・アラート通知
- 商圏分析マップ、e-Stat / RESAS / 不動産情報ライブラリ想定のAPI更新デモ
- 中途解約対応シナリオの入力シミュレーション
- 空室対策アシスタントによる後継テナント候補生成
- AIチャット相談デモ
- レポートプレビュー、PDF保存、画面キャプチャPDF保存
- 右上「デモ体験ナビ」から主要機能を順番に体験可能

## 公開方法
1. `index.html` と `assets/` をGitHubリポジトリ直下へ配置
2. GitHub Pagesを有効化
3. `https://<user>.github.io/<repo>/` で公開

## 本番化時の追加実装
- 認証・権限管理
- 物件DB/API接続
- 外部統計APIキー設定
- PDFテンプレートのサーバー側生成
- 操作ログ・承認ワークフロー・通知連携
